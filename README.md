# Coffee Shop E-Commerce

A full-stack web application built with **Spring Boot, Angular, and PostgreSQL**.

This project was developed as a practical exercise to build and connect a backend API with a modern frontend application.

---

## Tech Stack

- Java / Spring Boot  
- Angular
- PostgreSQL  
- Docker & Docker Compose
